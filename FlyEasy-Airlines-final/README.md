FlyEasy-Airlines is an online airline reservation system developed using Java, JDBC, and MySQL.  
The project demonstrates strong object-oriented design, modular architecture, and reliable database integration.

### Core Features
- Display available flights
- Book flight tickets with seat updates
- Transaction-based booking to maintain data consistency
- Robust error handling and input validation

### Technical Highlights
- DAO design pattern for clean separation of logic
- JDBC for secure database connectivity
- Transaction management using commit and rollback
- Modular and readable code structure

### How to Run the Project
1. Open the project in Eclipse or IntelliJ
2. Start MySQL server
3. Run the SQL file from the `sql` folder
4. Update database credentials in `DBConnection.java`
5. Run `Main.java`

### Innovation
Transaction-safe booking ensures seat availability remains accurate even during system failures.
